1. Dasar Teori Konsep Fungsi

Fungsi adalah unit program yang melakukan tugas tertentu dan mengembalikan sebuah nilai kepada pemanggilnya. Fungsi merupakan salah satu bentuk modularisasi yang memecah program besar menjadi unit-unit kecil yang lebih mudah dikelola.

Komponen utama fungsi:

  • Nama fungsi — pengenal unik
  • Parameter masukan — data yang diterima fungsi
  • Tipe nilai kembalian — tipe data yang dikembalikan
  • Badan fungsi — instruksi-instruksi di dalamnya
Fungsi vs Prosedur Fungsi selalu mengembalikan nilai (return). Prosedur melakukan aksi tanpa mengembalikan nilai. Dalam Python, keduanya ditulis dengan def, namun fungsi memiliki pernyataan return nilai.

def hitung_total(data):
    total = 0

def hitung_rata(data):

1. Sequential Search Linear Search

Sequential Search adalah metode pencarian yang memeriksa setiap elemen satu per satu dari awal hingga akhir array, sampai elemen yang dicari ditemukan atau semua elemen telah diperiksa.

Kompleksitas waktu: O(n) — semakin banyak data, semakin lama pencariannya secara proporsional.

Pseudocode — Sequential Search
FUNCTION SeqSearch(data : ARRAY, N : integer, kunci : integer) → integer
DEKLARASI
i : integer
ketemu : boolean
DESKRIPSI
i ← 1
ketemu ← FALSE
WHILE i ≤ N AND NOT ketemu DO
IF data[i] = kunci THEN
ketemu ← TRUE
ELSE
i ← i + 1
END IF
END WHILE
IF ketemu THEN
RETURN i { indeks posisi }
ELSE
RETURN -1 { tidak ditemukan }
END IF
END FUNCTION
sequential_search.py
def seq_search(data, kunci):
    """Mencari kunci dalam data, kembalikan indeks atau -1."""
    for i in range(len(data)):
        if data[i] == kunci:
            return i   # ditemukan, kembalikan indeks
    return -1             # tidak ditemukan

data = [15, 42, 8, 73, 27, 55]
kunci = 73
hasil = seq_search(data, kunci)

if hasil != -1:
    print(f"Kunci {kunci} ditemukan di indeks {hasil}")
else:
    print(f"Kunci {kunci} tidak ditemukan")

2. Binary Search

Binary Search hanya dapat diterapkan pada array yang sudah terurut. Cara kerjanya: bandingkan kunci dengan elemen tengah, lalu eliminasi setengah array yang tidak mungkin berisi kunci.

Kompleksitas waktu: O(log n) — jauh lebih cepat dari sequential search untuk data besar.

Pseudocode — Binary Search
FUNCTION BinSearch(data : ARRAY, N : integer, kunci : integer) → integer
DEKLARASI
kiri, kanan, tengah : integer
ketemu : boolean
DESKRIPSI
kiri ← 1
kanan ← N
ketemu ← FALSE
WHILE kiri ≤ kanan AND NOT ketemu DO
tengah ← (kiri + kanan) div 2
IF data[tengah] = kunci THEN
ketemu ← TRUE
ELSE IF kunci < data[tengah] THEN
kanan ← tengah - 1 { cari di kiri }
ELSE
kiri ← tengah + 1 { cari di kanan }
END IF
END WHILE
IF ketemu THEN RETURN tengah ELSE RETURN -1
END FUNCTION
binary_search.py
def bin_search(data, kunci):
    """Binary search — data harus sudah terurut ascending."""
    kiri = 0
    kanan = len(data) - 1

    while kiri <= kanan:
        tengah = (kiri + kanan) // 2
        if data[tengah] == kunci:
            return tengah
        elif kunci < data[tengah]:
            kanan = tengah - 1   # cari di setengah kiri
        else:
            kiri = tengah + 1    # cari di setengah kanan

    return -1   # tidak ditemukan

data = [8, 15, 27, 42, 55, 73, 89]  # sudah terurut
print(bin_search(data, 42))   # → 3
print(bin_search(data, 100))  # → -1

1. Bubble Sort

Bubble Sort bekerja dengan cara membandingkan dua elemen yang bersebelahan dan menukarnya jika urutannya salah. Proses ini diulang terus sampai tidak ada pertukaran yang terjadi.

Nama "bubble" karena elemen terbesar secara bertahap "menggelembung" ke akhir array pada setiap pass.

Pseudocode — Bubble Sort
PROCEDURE BubbleSort(INPUT/OUTPUT arr : ARRAY OF integer, INPUT n : integer)
DEKLARASI
i, j, temp : integer
tukar : boolean
DESKRIPSI
FOR i ← 1 TO n-1 DO
tukar ← FALSE
FOR j ← 1 TO n-i DO
IF arr[j] > arr[j+1] THEN
temp ← arr[j]
arr[j] ← arr[j+1]
arr[j+1] ← temp
tukar ← TRUE
END IF
END FOR
IF NOT tukar THEN BREAK { optimasi: hentikan jika sudah terurut }
END FOR
END PROCEDURE
bubble_sort.py
def bubble_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        tukar = False
        for j in range(n - 1 - i):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                tukar = True
        if not tukar:
            break   # sudah terurut, hentikan
    return arr

2. Selection Sort

Selection Sort bekerja dengan cara mencari elemen terkecil dari bagian array yang belum terurut, lalu menukarnya dengan elemen pertama dari bagian tersebut.

Pseudocode — Selection Sort
PROCEDURE SelectionSort(INPUT/OUTPUT arr : ARRAY OF integer, INPUT n : integer)
DEKLARASI
i, j, idx_min, temp : integer
DESKRIPSI
FOR i ← 1 TO n-1 DO
idx_min ← i { anggap elemen i adalah yang terkecil }
FOR j ← i+1 TO n DO
IF arr[j] < arr[idx_min] THEN
idx_min ← j
END IF
END FOR
IF idx_min ≠ i THEN { tukar jika perlu }
temp ← arr[i] ; arr[i] ← arr[idx_min] ; arr[idx_min] ← temp
END IF
END FOR
END PROCEDURE
selection_sort.py
def selection_sort(arr):
    n = len(arr)
    for i in range(n - 1):
        idx_min = i
        for j in range(i + 1, n):
            if arr[j] < arr[idx_min]:
                idx_min = j
        if idx_min != i:
            arr[i], arr[idx_min] = arr[idx_min], arr[i]
    return arr

1. Insertion Sort

Insertion Sort bekerja seperti menyusun kartu: setiap elemen baru disisipkan ke posisi yang tepat di bagian array yang sudah terurut.

Pseudocode — Insertion Sort
PROCEDURE InsertionSort(INPUT/OUTPUT arr : ARRAY OF integer, INPUT n : integer)
DEKLARASI
i, j, kunci : integer
DESKRIPSI
FOR i ← 2 TO n DO
kunci ← arr[i] { elemen yang akan disisipkan }
j ← i - 1
WHILE j ≥ 1 AND arr[j] > kunci DO
arr[j+1] ← arr[j] { geser elemen ke kanan }
j ← j - 1
END WHILE
arr[j+1] ← kunci { sisipkan di posisi yang tepat }
END FOR
END PROCEDURE
insertion_sort.py
def insertion_sort(arr):
    for i in range(1, len(arr)):
        kunci = arr[i]
        j = i - 1
        # Geser elemen yang lebih besar ke kanan
        while j >= 0 and arr[j] > kunci:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = kunci   # sisipkan kunci
    return arr

def urutkan(data):
    # detail insertion sort tersembunyi di sini
    for i in range(1, len(data)):

def input_nilai_valid(pesan):
    """Input nilai dengan validasi range 0–100."""
    while True:
        try:
            nilai = float(input(pesan))
            if 0 <= nilai <= 100:
                return nilai
            else:
                print("Error: Nilai harus antara 0–100!")
        except ValueError:
            print("Error: Masukkan angka yang valid!")

def input_valid(pesan, min_val=0, max_val=100):
    while True:
        try:
            val = float(input(pesan))
            if min_val <= val <= max_val: return val
            print(f"Masukkan nilai {min_val}–{max_val}!")
        except ValueError: print("Input tidak valid!")

def hitung_nilai_akhir(tugas, uts, uas):
    return tugas * 0.20 + uts * 0.35 + uas * 0.45

def tentukan_grade(nilai):
    if   nilai >= 85: return 'A'
    elif nilai >= 75: return 'B'
    elif nilai >= 60: return 'C'
    elif nilai >= 50: return 'D'
    else:             return 'E'

def tampilkan_hasil(tugas, uts, uas, akhir, grade):
    print("="*35)
    print(f"  Tugas  : {tugas:.1f}  × 20% = {tugas*0.2:5.2f}")
    print(f"  UTS    : {uts:.1f}  × 35% = {uts*0.35:5.2f}")
    print(f"  UAS    : {uas:.1f}  × 45% = {uas*0.45:5.2f}")
    print(f"  Nilai Akhir : {akhir:.2f}  Grade: {grade}")
    status = "LULUS" if grade != 'E' else "TIDAK LULUS"
    print(f"  Status : {status}")
    print("="*35)
